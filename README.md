# 🛒 ShopX – Mini Temu-Style E-commerce Frontend

A simple, modern e-commerce frontend inspired by platforms like Temu. This project demonstrates core shopping features such as product listings, search, category filtering, and a persistent cart using localStorage.

---

## 🚀 Features

* 🧾 Product listing grid
* 🔍 Live search functionality
* 🏷️ Category-based filtering (Fashion, Electronics)
* 🛒 Add-to-cart system
* 💾 Cart persistence using browser localStorage
* 📱 Responsive layout (basic)

---

## 🧱 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla)** – Functionality and state management

---

## 📁 Project Structure

```
shopx/
│── index.html     # Main HTML file
│── style.css      # Stylesheet
│── script.js      # App logic
```

---

## ⚙️ How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start browsing products and adding items to the cart

No installation or dependencies required.

---

## 🧠 How It Works

* Products are stored in a JavaScript array (`products`)
* Cart data is saved in `localStorage` so it persists after refresh
* Search updates results in real-time using input events
* Filters display products by category dynamically

---

## 📸 Demo Features

* Add items → updates cart count instantly
* Refresh page → cart remains saved
* Search bar → filters products live
* Category buttons → quick filtering
